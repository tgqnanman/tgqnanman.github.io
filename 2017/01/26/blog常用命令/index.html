blog常用命令

2017.01.26 啊其  热度 ℃ 条评论
  1. 主要配置文件：
  • 在根目录下的
    _config.yml (主配置)
    themes => theme => _config.yml (theme配置)
    source => _posts (md文件存放)
  1. 用到的git命令

  • 生成新的md文件： hexo n 'filename'

  • 整合静态文件： hexo g

  • 本地预览： hexo server

  • 上传到远端仓库： hexo deploy

  • 远端不显示可键入再deploy: hexo clear
