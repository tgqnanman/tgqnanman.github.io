Linux自己的常用命令

2019.02.05 Mr.Q  热度 ℃ 条评论
Order. Feature Command Consequent
a 查看内核版本命令 cat /proc/version
or
cat /etc/issue
b 查找指定文件名的文件 find -iname "test.txt"
c 显示文件大小 ls -lh
d 显示ssh客户端版本 ssh -V
e 输出当前工作目录 pwd
f 压缩为.gz的文件 gzip test.txt
g 解压.gz文件 gzip -d test.txt.gz
h 重启系统 shutdown -r now
or
reboot
i 显示运行进程的信息 ps -ef
j 査进程号 pgrep -l mongod
k 终止进程 kill -9 1600
or
killall -9 mongod
l 复制文件 cp -p file1 file2
m 重命名 mv -v file1 file2
n 查看多个文件内容 cat file1 file2
o 加上行号 cat -n file1
p 改变文件权限 chmod ug+rwx file.txt
or
chmod 600 file.txt
q 修改用户密码 passwd userName
r 超级用户切换用户 su userName
s 查看文件目录 find / -name nginx
