Mysql Server5.5的安装

Download MySQL Community Server：msi下载地址

  • 默认配置安装Mysql

  • 配置环境变量

  • C:\Program Files (x86)\MySQL\MySQL Server 5.5\bin（使得能够全局使用mysql）

  • 自启动mysql服务

  • 然后在service.msc将mysqlaaaaQ（这里是你配置的名字）服务设为自动，这样在开机将能直接通过下命令打开本地数据库（默认端口为3306，因为还安装了别本地数据库例如mongodb，就此指定3307端口打开mysql）
