Git的ssh

2018.02.15 Mr.Q  热度 ℃ 条评论

ssh登录git服务器（ssh登录）
用户 tgq
系统 centos6.9
【服务端】
首先登录tgq用户
然后
[tgq@instance-1 ~]$ cd ~
[tgq@instance-1 ~]$ mkdir .ssh
[tgq@instance-1 ~]$ vim .ssh/authorized_keys

【客户端】
本地找到C:\Users\TGQ.ssh
里面的id_rsa.pub

复制到【服务端】的.ssh/authorized_keys里面

【服务端】
然后给.ssh和authorized_keys加权限
[tgq@instance-1 ~]$ chmod 755 .ssh
[tgq@instance-1 ~]$ chmod 644 .ssh/authorized_keys

【TEST】
【客户端SSH登录linux的tgq用户-SUCCESS】
$ ssh tgq@35.196.180.56
Last login: Thu Apr 26 03:01:44 2018 from 211.97.6.123

给tgq用户加sudo权限，先用root用户登录然后
$ sudo vim /etc/sudoers
加入这条即可
tgqgit ALL=(ALL) NOPASSWD:ALL
